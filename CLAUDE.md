# Church App Template Notes

This repository has been sanitized for sharing as a template.

Replace these placeholders before building or deploying a new church app:

- Supabase URL and anon key in `src/supabase.js`
- Expo EAS project ID in `app.json` and `src/notifications.js`
- Church details, links, and social profiles in `src/data/index.js`
- Admin panel credentials in `admin/index.html`
- Bundle identifiers in `app.json`

Do not commit real service-role keys, admin passwords, or private deployment notes.
